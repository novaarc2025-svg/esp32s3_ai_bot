软件系统栈 (Software Stack)

  • 开发环境： 使用 ESP-IDF (VS Code 集成) 进行底层驱动编写。
  • 语音处理： 本地实现 Wake-word 检测（使用 ESP-Skainet），语音转文字（STT）通过网络 API 完成。
  • AI 核心： 接入大语言模型 (LLM, 如 GPT-4o 或 通义千问) API，实现生成式多轮对话。
  • 通信协议： 使用 MQTT 或 WebSocket 与后端进行双向实时数据交换。
